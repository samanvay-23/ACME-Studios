# API Routes (Acme AI)

## Clients
GET /clients → Fetch all clients  
GET /clients/{id} → Fetch single client  

## Campaigns
GET /campaigns → List campaigns  
POST /campaigns → Create new campaign  

## AI Ideas
GET /ideas → Fetch generated ideas  
POST /ideas → Generate new idea  

## Subscriptions
GET /subscriptions → Fetch subscription data  

## Analytics
GET /analytics/engagement → Campaign engagement metrics  
GET /analytics/revenue → Monthly revenue insights
